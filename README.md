# Portfolio Professionnel - Mehdi El Bidaoui

Un portfolio web moderne, dynamique et responsive conçu pour présenter mes compétences, mes projets académiques et mon expérience en tant qu'ingénieur en informatique et réseaux.


## 🚀 Fonctionnalités

*   **Design Moderne & Élégant** : Thème sombre (dark mode) avec une palette de couleurs soignée (Slate & Teal/Primary).
*   **Totalement Responsive** : Interface fluide s'adaptant à tous les écrans (téléphones, tablettes, ordinateurs).
*   **Téléchargement de CV** : Un bouton dédié pour télécharger directement le curriculum vitae au format PDF.
*   **Liens GitHub & Démo** : Accès direct aux dépôts des projets académiques et personnels.

## 🛠️ Stack Technique

Ce projet a été construit avec les technologies suivantes :
*   **React** (via Vite) - Bibliothèque typographique et d'interface utilisateur
*   **TypeScript** - Sur-ensemble typé pour JavaScript
*   **Tailwind CSS (v4)** - Framework CSS utilitaire pour le style rapide
*   **Lucide React** - Superbe bibliothèque d'icônes
*   **PostCSS** - Outil de transformation CSS

## 📂 Structure du projet

Les composants principaux se trouvent dans :
*   `src/components/Hero.tsx` : La section d'accueil avec l'animation 3D du profil.
*   `src/components/Experience.tsx` : L'historique des stages et l'expérience professionnelle.
*   `src/components/Projects.tsx` : La liste des projets avec les liens GitHub.
*   `src/components/Skills.tsx` : Les langages de développement, frameworks et certifications.

## ⚙️ Installation et Déploiement

Pour exécuter ce projet localement :

1.  **Cloner le dépôt**
    ```sh
    git clone https://github.com/votre-utilisateur/portfolio.git
    cd portfolio
    ```

2.  **Installer les dépendances**
    ```sh
    npm install
    ```

3.  **Lancer le serveur de développement**
    ```sh
    npm run dev
    ```

4.  **Compiler pour la production**
    ```sh
    npm run build
    ```
